Analysis of hacker attacks

It was March 2023 when a fresh virtual machine on Azure was set up. Its user sat down happy to work with their new remote environment but little did he know how exposed he has become...



The point of the study was to set up a virtual machine on Azure and monitor all the hacker attacks coming in. The results of this analysis are available on this website under respective links.


The information on these subpages is a quick insight into the most important observations regarding the plots. The more complete analysis can be found in the report.
